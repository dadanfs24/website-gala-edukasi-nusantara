# Website Gala Edukasi Nusantara (GEN)

Next.js (App Router) + Tailwind CSS v4 + pnpm.

## Run
```bash
pnpm install
pnpm dev
```

## Anti-error Tailwind v4
- Tailwind v4 PostCSS plugin: `@tailwindcss/postcss` (bukan `tailwindcss`).
- Proyek ini menghindari `@apply` untuk utility custom/variant agar tidak muncul error seperti sebelumnya.

## Pages
- `/` Home
- `/beasiswa` + detail
- `/artikel` + detail
- `/event` + detail
- `/dashboard/kampus` + contoh free/premium
- `/upcoming` advanced feature placeholder
